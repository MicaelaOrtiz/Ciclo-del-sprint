# Sistema de Pedidos

Aplicación web desarrollada para gestionar pedidos en un local de comida, implementando el ciclo de vida del software con metodología ágil.

##  Funcionalidades 
 Crear pedidos con nombre de cliente y productos
 Calcular el total automáticamente
 Visualizar la lista de pedidos
 Cambiar el estado del pedido:
  Pendiente
  En preparación
  Entregado


##  Tecnologías utilizada 
Backend: Python con Flask
Frontend: HTML, CSS y JavaScript
Control de versiones: Git
Gestión del proyecto: Trello


##  Gestión del proyecto

Se utilizó un tablero en Trello para organizar el sprint, dividiendo el trabajo en:

Backlog
To Do
En progreso
Testing
Done


## Instalación y ejecución

1. Clonar el repositorio:

```
git clone https://github.com/tu-usuario/sistema-pedidos.git
```
2. Entrar a la carpeta del proyecto:

cd sistema-pedidos
 Instalar dependencias:
pip install flask

Ejecutar la aplicación:
python app.py

 Abrir en el navegador:
http://127.0.0.1:5000

## Flujo de trabajo (Git)

Se utilizó una estrategia de ramas:
`feature/*` para nuevas funcionalidades
 `develop` para integración
 `release/1.0.0` para versión final
 `master` como rama principal


## Testing

Se realizaron pruebas manuales para validar:

 Creación de pedidos

## Metodología

Se aplicó un enfoque ágil tipo Scrum:

 Definición de requerimientos
 Creación de casos de uso
 Planificación en Trello
 Desarrollo incremental
 Demo del sistema
 Feedback del cliente
 Nuevo sprint con mejoras
